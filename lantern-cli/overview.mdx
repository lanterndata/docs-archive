[Lantern CLI](https://github.com/lanterndata/lantern_extras#lantern-index-builder) is collection of tools written in Rust which will help you:

- Generate index for your data blazingly fast without overloading the database server
- Generate embeddings for your Postgres data on both CPU and GPU
- Run daemon to continuously generate embeddings for the newly inserted rows asyncronouesly
